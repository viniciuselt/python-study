Algoritmos sequenciais. Dados, Variáveis e Comandos de Entrada e Saída

Os algoritmos sequenciais são uma sequência de instruções executadas em ordem para realizar uma determinada tarefa. Em Python, podemos escrever algoritmos sequenciais que manipulam dados e variáveis. Os dados podem ser de vários tipos em Python, incluindo números, strings, listas, dicionários e muito mais. As variáveis são usadas para armazenar dados e podem ser declaradas e atribuídas um valor em Python. Os comandos de entrada e saída em Python são usados para interagir com o usuário. O comando "input" é usado para receber dados do usuário, enquanto o comando "print" é usado para exibir dados na tela.

Um exemplo de algoritmo sequencial em Python seria um programa que calcula a área de um círculo. Ele pode receber o valor do raio do usuário usando o comando "input", calcular a área usando a fórmula apropriada e exibir o resultado usando o comando "print". Em resumo, os algoritmos sequenciais em Python usam dados e variáveis para executar tarefas específicas e usam comandos de entrada e saída para interagir com o usuário.

Os cálculos de expressões aritméticas funcionam da mesma maneira que na matemática tradicional. Ou seja, a ordem de precedência dos operadores deve ser respeitada da mesma maneira que você respeita ao fazer um cálculo na ponta do lápis.

VARIÁVEIS, DADOS E SEUS TIPOS

De acordo com nossa bibliografia, citamos três tipos de variáveis denominados tipos primitivos de dados:

  • Numérico – é um tipo que serve para representar qualquer número, sejam valores inteiros, ou valores com casas decimais (também chamados de ponto flutuante). Uma variável deve ser declarada como numérica quando operações aritméticas precisarem ser realizadas com elas;
  • Caractere – é um tipo que serve para representar letras, caracteres especiais, acentuações, e até mesmo podem ser utilizados para números, quando estes não servirem para operações aritméticas, por exemplo;
  • Literal/booleano – é um tipo que serve para representar somente dois estados lógicos: verdadeiro ou falso (1 ou 0).

No código a seguir estamos criando duas variáveis (não vamos tratar do tipo ainda). A primeira tem nome disciplina, e armazena o nome da nossa disciplina. A outra se chama nota e armazena uma nota tirada nessa disciplina.

nota = 6.5
disciplina = 'Lógica de programação e algoritmos'

print(nota)
print(disciplina)

print('Disciplina:', disciplina, 'A nota é: ', nota)

Observe que o sinal de igual é o que indica que o conteúdo colocado à direita será armazenado na variável colocada à esquerda. Em programação, o sinal de igual significa atribuição, ou seja, o valor é atribuído à variável.
